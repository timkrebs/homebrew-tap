# Homebrew Tap for custos

This is a [Homebrew](https://brew.sh) tap for [custos](https://github.com/timkrebs/custos) — the missing `terraform plan` for HashiCorp Vault policies.

## Installation

```bash
brew install timkrebs/tap/custos
```

## Update

```bash
brew upgrade custos
```

## Formula

The formula in `Formula/custos.rb` is **auto-updated by GoReleaser** on each release. Manual edits will be overwritten.
